# Canon

Crystallized knowledge. Everything here has a clear type, stable identity, and linked context.

## Types

| Type | Folder | Emoji | What lives here |
|------|--------|-------|----------------|
| person | People/ | 👤 | Everyone mentioned in voice memos |
| event | Events/ | 📅 | Things that happened |
| action | Actions/ | 🎯 | Tasks, intentions, things to do |
| decision | Decisions/ | ⚖️ | Choices made |
| belief | Beliefs/ | 🪨 | Things held to be true |
| project | Projects/ | 📁 | Larger efforts |
| place | Places/ | 📍 | Locations with context |
| organization | Organizations/ | 🏢 | Companies, groups |

## Rules

- Every entry has typed frontmatter (type, name, created, source)
- Every entry has an emoji heading matching its type
- Wikilinks connect everything — use them generously
- Provenance is mandatory — every fact traces to a source
- Locked fields cannot be changed by AI agents
